 Movie Ticket Booking System (ADBMS)

 Project Overview
The Movie Ticket Booking System is a database-driven application developed as part of the Advanced Database Management Systems (ADBMS) course.  
The system is designed to automate the process of movie ticket booking by managing movies, show timings, seats, users, and bookings efficiently using relational database concepts.

This project focuses on applying core ADBMS concepts in a real-world scenario.

---

Objectives
- To design an efficient database for a movie ticket booking system
- To apply database normalization and constraints
- To manage bookings, users, and movie schedules
- To demonstrate practical use of ADBMS concepts



 Features
- View available movies
- View show timings
- Seat selection
- Movie ticket booking
- Booking management
- Database-driven operations

---

Technology Stack
 Backend SqlServer
 Frontend: C# (Window Forms)
Database: SQL Server 
Tools: Visual Studio, SQL Server Management Studio


 Database Concepts Used
- Entity Relationship (ER) Diagram
- Normalization (1NF, 2NF, 3NF)
- Primary Keys & Foreign Keys
- Constraints
- Stored Procedures

---

 Database Tables (Example)
- Movies
- Shows
- Seats
- Bookings
